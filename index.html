Quick Start Examples

# Basic usage – 3 auto commits
$ fahim-autocommit --count 3

# With custom delay (2–5 seconds between commits)
$ fahim-autocommit --count 5 --min-delay 2 --max-delay 5

# Custom commit message template
$ fahim-autocommit --count 2 --message "docs: update README + automation"

# Dry-run: see what would happen
$ fahim-autocommit --count 4 --dry-run
