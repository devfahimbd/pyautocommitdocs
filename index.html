Installation Guide

$ pip install fahim-autocommit
$ pip install --upgrade fahim-autocommit
$ pip uninstall fahim-autocommit
$ pip show fahim-autocommit
Requires Python 3.8+ and Git installed globally.

Quick Start Examples

$ fahim-commit --count 10
$ fahim-commit --count 10 --min-delay 1
$ fahim-commit --count 10 --max-delay 3
$ fahim-commit --count 20 --min-delay 1 --max-delay 2
$ fahim-commit --count 5 --message "Commit {iteration} at {time}"
$ fahim-commit --count 5 --message "Updated {file} on {time}"
$ fahim-commit --count 5 --dry-run
$ fahim-commit --count 25 --min-delay 1 --max-delay 3 --message "Modified {file} at {time}"
$ fahim-commit --help
